# HTML CSS Viewer

A powerful online tool for live HTML and CSS code editing and preview. Try it now at [htmlcssviewer.github.io](https://htmlcssviewer.github.io/)

[![Website](https://img.shields.io/website?url=https%3A%2F%2Fhtmlcssviewer.github.io)](https://htmlcssviewer.github.io/)
[![GitHub license](https://img.shields.io/github/license/htmlcssviewer/htmlcssviewer.github.io)](https://github.com/htmlcssviewer/htmlcssviewer.github.io/blob/main/LICENSE)

## ✨ Features

- 🚀 **Real-time Preview**: See your HTML and CSS changes instantly as you type
- 💻 **No Installation**: Works directly in your browser, no setup required
- 🎨 **Live CSS Editing**: Test and experiment with styles in real-time
- 📱 **Responsive Testing**: Preview your design across different screen sizes
- 🔒 **Privacy First**: No data collection, everything stays in your browser
- 🆓 **Free & Open Source**: Always free to use, open for contributions

## 🚀 Quick Start

1. Visit [htmlcssviewer.github.io](https://htmlcssviewer.github.io/)
2. Write your HTML code in the editor
3. Add your CSS styles
4. See the results instantly in the preview panel
